# 🌐 Metaverse Business Engagement & Revenue Dashboard

## 📊 Project Overview
This project presents an interactive **Power BI dashboard** designed to analyze business performance and user engagement within a metaverse ecosystem. The dashboard provides insights into user behavior, revenue trends, device adoption, and platform engagement metrics.

The goal of this project is to help stakeholders understand key performance indicators (KPIs) and make data-driven decisions to optimize virtual platform growth.

---

## 📁 Dataset Description
The dataset used in this project contains simulated metaverse platform data, including:

- User activity and engagement
- Virtual real estate views
- NFT purchases
- Device usage (VR, AR, Desktop)
- Session duration
- User demographics (region, avatar type)
- Satisfaction ratings
